# Amőba
# A játék alapjai
Maga a játék két játékos között zajlik az egyik mint felhasználó a másik mint a Mesterséges Intelligencia
# A játék szabályai
A felhasználó támad, a mesterséges Intelligencia védekezik.
A két fél célja, hogy egyszerre megakadályozzák ellenfelüket, valamint saját szimbólumokból kigyűjtsék az 5 darabot.
# A játék menete
A felhasználó elkezdi a játékot és a pája valamely pontjára lehelyez egy x-et.
A Mesterséges Intelligencia erre válaszolva a pálya valamely pontjára lehelyez egy o-t.
Majd elkezdődik a játék ami addig folytatódik amig a két fél közül az egyik eléri az 5 egymás melletti szimbólumot.
Ezután a játék végetér, a pontokat a program elraktározza, és kezdődhet is a következő játék.
